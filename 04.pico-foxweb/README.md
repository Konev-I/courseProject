# Курсовой проект, вариант №2

## Сборка/запуск

- Сборка

~~~
sudo make install
~~~

- Запуск

~~~
./PICOFoxweb
~~~

- Проверка 
Для проверки аутентификации и авторизации пользователя на сервере, в браузере нужно обращаться по uri начинающегося с "/private", например:

~~~
http://127.0.0.1:8000/private/index.html
~~~
